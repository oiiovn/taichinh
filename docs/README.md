# Tài liệu Hệ thống

Hệ thống tài liệu gồm **5 bộ tài liệu chính**:

| # | Tài liệu | File | Mục đích |
|---|----------|------|----------|
| 1 | **Product Doc** | [PRODUCT_DOC.md](./PRODUCT_DOC.md) | Sản phẩm: scope, user stories, luồng nghiệp vụ, yêu cầu chức năng & NFR. |
| 2 | **Technical Doc** | [TECHNICAL_DOC.md](./TECHNICAL_DOC.md) | Kỹ thuật: stack, kiến trúc, API, DB, triển khai, test. |
| 3 | **Financial Logic Doc** | [FINANCIAL_LOGIC_DOC.md](./FINANCIAL_LOGIC_DOC.md) | Logic tài chính: công thức, ngưỡng, định nghĩa, nguồn dữ liệu. |
| 4 | **AI Learning Doc** | [AI_LEARNING_DOC.md](./AI_LEARNING_DOC.md) | AI/Học máy: payload, model, learning loop, chất lượng. |
| 5 | **Security Doc** | [SECURITY_DOC.md](./SECURITY_DOC.md) | Bảo mật: xác thực, phân quyền, dữ liệu nhạy cảm, ứng phó sự cố. |
| 6 | **Behavior Intelligence Doc** | [BEHAVIOR_INTELLIGENCE_DOC.md](./BEHAVIOR_INTELLIGENCE_DOC.md) | Stack 9 tầng hành vi: Identity → Behavior → Trust → Projection. |

---

## Cập nhật

- Điền nội dung cụ thể vào từng file (thay placeholder [ … ]).
- Mỗi doc có phần **Changelog** ở cuối — cập nhật khi thay đổi.
- Tham chiếu chéo giữa các doc khi cần (ví dụ: Technical ↔ Financial Logic, AI Learning ↔ Product).
